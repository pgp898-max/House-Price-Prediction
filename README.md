# House Price Prediction

## 📌 Project Overview
This project builds a machine learning model to predict house prices based on features like area, number of bedrooms, and bathrooms.

## 🎯 Problem Statement
To predict the price of a house using regression techniques based on given features.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🔍 Features Used
- Area
- Bedrooms
- Bathrooms

## 🤖 Model Used
Linear Regression

## 📊 Model Evaluation
The model performance is evaluated using Mean Squared Error (MSE).

## 📈 Visualization
Actual vs Predicted prices graph is used to evaluate model performance.

## 📌 Conclusion
The model can predict house prices reasonably based on input features, but performance can be improved with more data and advanced algorithms.
